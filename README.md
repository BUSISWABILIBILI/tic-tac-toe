# Tic Tac Toe

A modern Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features

- Interactive Tic Tac Toe gameplay
- Custom player names
- Score tracking
- Restart game functionality
- Winning cell highlights
- Responsive modern UI
- Built using factory functions and the module pattern

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Concepts Practiced

This project focuses on:

- Factory Functions
- Module Pattern (IIFE)
- DOM Manipulation
- Event Listeners
- Game Logic
- Encapsulation
- Closures
- Responsive Design
